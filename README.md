# Venkata Siddhardha Morla — Portfolio

A dark editorial-style personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🚀 Live Demo
[View Portfolio](https://venkatasiddhardha.github.io/portfolio/)

## ✨ Features
- Custom animated cursor
- Scroll-reveal animations
- Responsive design (mobile + desktop)
- Scrolling tech stack marquee
- Hover-reveal skill cards
- Project showcase
- Experience & Education timeline
- Contact section

## 🛠 Tech Stack
- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (IntersectionObserver, requestAnimationFrame)
- Google Fonts (Bebas Neue, DM Mono, Fraunces)

## 📁 Structure
```
portfolio/
├── index.html       # Main portfolio file (all-in-one)
└── README.md
```

## 🌐 Deploy on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://<your-username>.github.io/portfolio/`
